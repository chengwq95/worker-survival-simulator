# 🏢 打工人生存 Simulator

一款轻松搞笑的中文办公室生存小游戏：在完成 KPI、回复消息、应对突发事件与躲避老板巡逻之间，努力活过每一个工作日。

## 在线游玩

[点击开始游戏](https://worker-survival-simulator.pages.dev/)

## 玩法

- 点击「努力工作」推进任务、完成 KPI、获得工资。
- 点击「摸鱼」回复心情，但老板巡逻时被抓会受到处罚。
- 留意老板预警、突击检查和消息弹窗，及时作出选择。
- 使用商店、技能与副业，提升存活和赚钱能力。
- 与同事互动，解锁不同增益、成就与结局。

## 操作

| 操作 | 作用 |
| --- | --- |
| 💼 努力工作 | 推进当前任务，消耗心情 |
| 🐟 摸鱼 | 恢复心情，但有被老板抓到的风险 |
| 📈 摸鱼副业 | 高风险、高收益 |
| 💬 立即回复 | 处理甲方与工作消息 |
| 🛒 商店 | 购买永久增益 |
| 🔊 音效 | 开启或关闭游戏音效 |

## 本地运行

这是纯单文件 HTML 游戏，无需安装依赖。下载仓库中的 index.html 后，直接用浏览器打开即可游玩。
